# EnglishCentral.com Automation Suite

A professional Selenium-based automation project for EnglishCentral.com, implementing high-quality QA standards and the Page Object Model (POM) design pattern.

## 🚀 Project Overview
This project automates critical end-to-end user flows on EnglishCentral, ensuring the reliability of core learning features. It includes a comprehensive suite of automated tests, manual test documentation, and API verification.

## 🛠️ Tech Stack
- **Language**: Java 11
- **Automation Tool**: Selenium WebDriver (4.15.0)
- **Test Runner**: TestNG (7.8.0)
- **Build Tool**: Maven
- **Design Pattern**: Page Object Model (POM)
- **Dependency Management**: WebDriverManager (for automatic driver setup)

## ✨ Key Automated Features
- **Smart Navigation**: Verification of landing pages and active user sessions.
- **Course Discovery**: Automated filtering by difficulty levels (**Beginner, Intermediate, Advanced**).
- **Interactive Learning**:
    - Automated course and unit selection.
    - Video player functionality testing (Playback, Resolution switching to 360p).
    - Subtitle/Transcript toggle verification.

## 📁 Project Structure
- `src/main/java/pages/`: Contains Page Objects for modular and maintainable web interactions.
- `src/test/java/tests/`: TestNG test classes covering the automation logic.
- `pom.xml`: Configuration and dependency management.

## 📋 QA Documentation Package
Beyond automation, this project provides:
- **Manual Test Suite**: Detailed Excel/CSV test cases covering all modules.
- **Bug Report Log**: Professional-grade bug tracking with UI/UX and functional insights.
- **API Collection**: Postman environment for one-click speech and video AI endpoint verification.
